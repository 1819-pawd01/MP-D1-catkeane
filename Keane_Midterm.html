CSS Cheat Table
Property Name Property Values Property Description
color color name The color property changes the color of the text.
font-family serif, sans-serif, cursive, fantasy, inherit, or monospace, and a font name. The font-family specifies the font for an element. It can hold several font names as a "fallback system." In case the browser does not support the font.
text-decoration overline, line-through, underline, or underline overline The text-decoration property specifies the decoration added to the text.
text-indent px, em, or % The text-indent property specifies the indentation of the first line in a text-block.
height auto, px, or pt The height property sets the height of the element.
width auto, px, or pt The width property sets the width of the element.
text-allign center, left, right, or justify The text-allign property specifies the horizontal allignment of the text in the element.
text-transform uppercase, lowercase, and capitalize The text-transform property controls the capitalization of the text.
background-color color name The background-color property sets the background of an element.
font-variant small-caps The font-variant property specifies whether or not the text should be displayed in a small-caps font.
 
Attribute Cheat Table
Attribute Name Attribute Values Attribute Description Elements
href URLs The href attribute specifies the URL of the link the page goes to. <a> or <link>
src URLs The src attribute specifies the URL of the external resource. <audio>, <embed>, <iframe>, <img>, <input>, <script>, <source>, <track>, or <video>
alt Text The alt attribute specifies an alternate text for an image, if the image cannot be displayed. <img>
target _blank, _parent, _self, or _top The target attribute specifies where to open the linked document <a>
title Text The title attribute specifies extra information about an element. Can be used in all elements
